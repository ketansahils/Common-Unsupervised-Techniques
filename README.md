# Common-Unsupervised-Techniques
Contains code to run and visualize techniques like Clustering, PCA on publicly available data. Comments are also provided to make it more interactive and useful. Some notebooks finish with classification.

* http://archive.ics.uci.edu/ml/datasets/Wine
* http://yann.lecun.com/exdb/mnist/
* https://www.python.org/downloads/release/python-360/
* https://matplotlib.org/
* https://www.numpy.org/
* https://pandas.pydata.org/
